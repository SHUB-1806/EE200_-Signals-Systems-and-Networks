### EE200: Signals, Systems and Networks – Course Project

A course project focused on frequency-domain signal processing for both images and audio signals.

**Part 1: Frequency Mixer – "Beauty and the Blur"**

* Analyzed images using the 2D Fourier Transform.
* Visualized magnitude and dB spectra.
* Verified Fourier transform properties through image rotation.
* Designed a frequency mixer to create hybrid images by combining low-frequency components from one image with high-frequency components from another.

**Part 2: Frequency De-mixer – "Unwanted Solo"**

* Performed spectral analysis of a corrupted audio signal.
* Identified unwanted instrumental frequency components using FFT and PSD analysis.
* Designed and evaluated filters using Bode plots and pole-zero analysis.
* Recovered the original audio by suppressing the interfering frequencies.

**Tools:** Python, NumPy, SciPy, OpenCV, Matplotlib, Librosa
